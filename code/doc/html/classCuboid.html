void Cuboid::generateCuboid ( )

Generates a cuboid at #height distance from the base.

I wanted to have highlighted edges for the cubes and drawing a new wire cube seemed to be the simplest solution 

{
    glColor3f(red, green, blue);
    glPushMatrix();

    glTranslatef(xTrans, yTrans, zTrans);
    glScalef(xScale, yScale, zScale);
    glutSolidCube(1.0);

    glColor3f(0.0, 0.0, 0.0); 
    glutWireCube(1.0);

    glPopMatrix();
}
